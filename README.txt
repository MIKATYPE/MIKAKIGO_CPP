注意事項

１．

Visual Studio 2022
Visual C++ 2022
64ビット版で動作確認済です

２．

Visual C++がインストールされていないパソコンで動作させるにはVisual C++ のランタイムライブラリをインストールする必要があります。

３．

Visual Studioでコンパイルするときにはプロジェクト名をMIKAKIGOにして下さい。

４．

このコースコードでは成績ファイル書き込み時にエラーがあってもダイアログを表示しません。

５．

キーボードの種類によっては記号の配列が異なる場合があります。

６.

チルダー記号 「~」 は 「～」 で表示されます。

７.

バックスラッシュキー「＼｣キーを押すと「￥」記号が表示されますがバグではありません。
